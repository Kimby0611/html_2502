screen 객체

.width >> 화면의 너비
.height >> 화면의 높이
.availWidth >> 화면의 너비(작업표시줄 제외)
.availHeight >> 화면의 높이(작업표시줄 제외)
.colorDepth >> 모니터에서 표현할 수 있는 컬러비트

location 객체

.href >> 참조 주소 설정, URL반환
.hash >> URL의 해시값 반환(#)
.hostname >>호스트 이름 반환 및 설정
.host >>호스트 이름과 포트 번호 반환
.protocol >>프로토콜 반환
.search >>쿼리 반환
.reload >>새로고침

history 객체

.back >> 이전 사이트
.forward >> 다음 사이트
.go(이동 숫자) >> 이동 숫자에 넣은 값만큼 페이지 이동 (-는 이전)
.length >> 방문 기록에 저장된 목록의 개수 반환
